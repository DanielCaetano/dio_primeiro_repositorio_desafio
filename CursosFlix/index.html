Caminho das pedras

  1. 1) Aprenda lógica de programação: Que tal pegar um cursinho de Portugol usando o Portugol Studio?
  2. 2) Aprenda agora lógica de programação na linguagem de programação que você deseja se tornar um expert. Você pode encontrar tal material aqui na Dio, na Caelum, na Alura, no YT, e até na internet.
  3. 3) Tente ver qual o melhor framework dessa linguagem, teste-os para fazer algo básico, veja qual você acha mais fácil, e use-o para os próximos passos
  4. 4) Aprenda como fazer um Rest API com essa linguagem de programação
  5. 5) Aprenda a criar uma aplicação CRUD (Criar, Ler, Atualizar, e Deletar), como na Rest API, mas agora utilizando um banco de dados
  6. 6) Que tal agora começar à desenvolver seus pequenos projetinhos pessoais, para consolidar todas as skills que adquiriu até aqui?!
  7. 7) Após essa consolidação, você pode decidir que caminho quer trilhar: se desenvolvimento de APPs, Tester , Jogos, Big Data, Mobile, etc.
  8. 8) Aprenda a usar Git, e matenha seus projetinhos upados no seu perfil no GitHub. Ele é seu cartão de visita, também.
